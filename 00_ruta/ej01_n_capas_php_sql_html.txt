                    02   USE DB_PROJECTO ;
                    03 
                    04   CREATE TABLE ROLES (
                    05     rol_codigo INT NOT NULL AUTO_INCREMENT,
                    06     rol_nombre VARCHAR(45) NOT NULL,
                    07     PRIMARY KEY (rol_codigo)
                    08   ) ENGINE = InnoDB;
                    08 
                    09   CREATE TABLE USUARIOS (
                    10     rol_codigo INT NOT NULL,  
                    11     usuario_codigo INT NOT NULL AUTO_INCREMENT,
                    12     usuario_nombres VARCHAR(100) NOT NULL,
                    13     usuario_apellidos VARCHAR(100) NOT NULL,
                    14     usuario_identificador VARCHAR(20) NOT NULL,
                    15     usuario_email VARCHAR(50) NOT NULL,
                    16     usuario_pass VARCHAR(200) NOT NULL,
                    17     usuario_estado TINYINT NOT NULL,
                    18     PRIMARY KEY (usuario_codigo),
                    19     INDEX ind_usuario_rol (rol_codigo ASC),
                    20     CONSTRAINT fk_usuario_rol
                    21       FOREIGN KEY (rol_codigo)
                    22       REFERENCES ROLES (rol_codigo)
                    23       ON DELETE CASCADE
                    24       ON UPDATE CASCADE
                    25   ) ENGINE = InnoDB;

|-- 06.03.  Codificar el Modelo de Conexión a la Base de Datos: -----------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Escrbir en el archivo 'models/DataBase.php' el siguiente código (Modelo de Conexión PDO): 
                    
                    01   <?php
                    02     class DataBase {    
                    03       public static function connection(){        
                    04         $hostname = "localhost";
                    05         $port = "3306";
                    06         $database = "db_projecto";
                    07         $username = "root";
                    08         $password = "";
                    09         $pdo = new PDO("mysql:host=$hostname;port=$port;dbname=$database;charset=utf8",$username,$password);
                    10         $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
                    11         return $pdo;
                    12       }
                    13     }
                    14   ?>
            
                
                EXPLICACIÓN:   
                    
                    ⦿   Descripción: El Modelo de Conexión a la Base de Datos en 'MySQL' es de la Clase 'PDO'.
                    ⦿   Las líneas 01 y 14, son para crear un bloque de código 'php'.
                    ⦿   En la línea 02, se crea una Clase con el nombre 'Database' y cierra el bloque en la línea 13.
                    ⦿   En la línea 03, se crea una función (o método) pública(o) con el nombre 'connection()', de forma estática (Se usa directamente 
                         el método al llamar la Clase, no es necesario crear un objeto) y cierra el bloque en la línea 12. Esta función retorna un
                         objeto de la Clase 'PDO' con la cadena de conexión a la base de datos.
                    ⦿   De las líneas 04 a la 08, se almacenan en variables los datos solicitados en una conexión a la Base de Datos 'MySQL'.
                    ⦿   En la línea 09, se crea una instancia de la Clase 'PDO' y como parámetros se le pasa al Constructor de la Clase la sintaxis de 
                         la cadena de conexión a la Base de Datos 'MySQL'.
                    ⦿   En la línea 10, se utiliza el método 'setAttribute()' que, recibe los posibles errores que surjan al intentar conectar 
                         con la Base de Datos 'MySQL'.
                    ⦿   En la línea 11, se retorna un objeto '$pdo' de la Clase 'PDO' con la cadena de conexión a la base de datos.


            02. Escrbir en el archivo 'index.php' el siguiente código: 

                    01   <?php

                EXPLICACIÓN:   
                    
                    ⦿   Descripción: Este código permite verificar si hay algún problema con la conexión a la Base de Datos.
                    ⦿   Las líneas 01 y 04, son para crear un bloque de código 'php'.
                    ⦿   En la línea 02, se llama al archivo (tipo Clase) de conexión a la Base de Datos.
                    ⦿   En la línea 03, se crea una variable (objeto) para recibir el objeto '$pdo' que, tiene la conexión a la Base de Datos.

|-- 06.04.  Codificar el Inicio del Proyecto y el Enrutador a Controladores -----------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Modificar al archivo 'index.php' con el siguiente código:
                    
                    01   <?php
                    02     require_once "modelos/DataBase.php";
                    03     if (!isset($_REQUEST['c'])) {
                    04       require_once "controladores/Usuarios.php";
                    05       $controlador = new Usuarios;
                    06       $controlador->rolRegistrar();
                    07     } else {        
                    08       $controlador = $_REQUEST['c'];
                    09       require_once "controladores/" . $controlador . ".php";
                    10       $controlador = new $controlador;        
                    11       $accion = isset($_REQUEST['a']) ? $_REQUEST['a'] : 'main';
                    12       call_user_func(array($controlador, $accion));
                    13     }
                    14   ?>

                
                EXPLICACIÓN:   
                    
                    ⦿   Descripción: El código de este archivo permite iniciar la aplicación y enrutar al controlador, dependiendo de los datos 
                         que se envíen por la 'URL', donde 'c' es el controlador (Clase) y 'a' es la acción (función o método).
                    ⦿   Las líneas 01 y 14, son para crear un bloque de código 'php'.
                    ⦿   En la línea 02, se llama al archivo (tipo Clase) de conexión a la Base de Datos.
                    ⦿   En la línea 03, 07 y 13, se crea una estructura de toma de decisión 'if-else', que sirve para saber cuál controlador (c) y
                         acción (a) se debe buscar en la carpeta 'controllers/'.
                    ⦿   En la línea 03, la variable super global '$_REQUEST['c']' busca en la 'URL' el valor de la variable 'c', mientras que la 
                         función 'isset($_REQUEST['c'])' determina si tiene un valor (verdadero) o por el contrario es 'NULL' (falso). Con '!' se
                         niega toda la expresión. En síntesis, si 'c' no existe en la 'URL', el resultado de la expresión es 'verdadero', lo que indica
                         que se ejecutan las líneas 04, 05 y 06; si 'c' existe en la 'URL' y tiene valor, el resultado de la expresión es 'falsa', lo
                         que indica que se ejecutan las líneas 08, 09, 10, 11 y 12.
                    ⦿   En la línea 04, se lama el archivo (Clase) 'Landing.php'
                    ⦿   En la línea 05, se crea una instancia de la Clase (objeto) de 'Landing' 
                    ⦿   En la línea 06, se llama a la función (o método) 'main' de la clase.
                    ⦿   En la línea 08, se pasa el valor de la variable 'c' a la variable '$controller'
                    ⦿   En la línea 09, se llama el archivo (Clase) 'controllers/' . $controller . '.php'
                    ⦿   En la línea 10, se crea una instancia de la Clase (objeto) 'controllers/' . $controller . '.php'
                    ⦿   En la línea 11, se usa un 'if terniario' para pasarle el valor de la variable 'a' a la variable '$action'. Si la variable 'a'
                         tiene valor, le pasa el valor a '$action', de lo contrario su valor será 'main'
                    ⦿   En la línea 12, se pasan como parámetros a la función 'array()' el valor de las variables '$controller' y '$action'. Mientras
                         que, con 'call_user_func()', se busca un archivo en la carpeta 'controllers/' con una Clase que se llame igual al valor de 
                         '$controller' y una función que se llame igual al valor de '$action'.
