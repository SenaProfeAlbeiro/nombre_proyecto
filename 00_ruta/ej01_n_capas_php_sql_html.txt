|-- 06.03.  Codificar el Modelo de Conexión a la Base de Datos: -----------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Escrbir en el archivo 'models/DataBase.php' el siguiente código: 
                    
                    01   <?php
                    02     class DataBase {    
                    03       public static function connection(){        
                    04         $hostname = "localhost";
                    05         $port = "3306";
                    06         $database = "db_project";
                    07         $username = "root";
                    08         $password = "";
                    09         $pdo = new PDO("mysql:host=$hostname;port=$port;dbname=$database;charset=utf8",$username,$password);
                    10         $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
                    11         return $pdo;
                    12       }
                    13     }
                    14   ?>
            
                ┌ ------------------------------------------------------------------------------------------------------------------------------------ ┐
                |                                                                                                                                      |
                  EXPLICACIÓN:   
                    
                    ⦿   El código de este archivo, representa el Modelo de Conexión a la Base de Datos en 'MySQL' a través de la Clase 'PDO'.

                |--------------------------------------------------------------------------------------------------------------------------------------|                

                    A.   Las líneas 01 y 14, son para crear un bloque de código 'php'.

                    B.   En la línea 02, se crea una Clase con el nombre 'Database' y cierra el bloque en la línea 13.

                    C.   En la línea 03, se crea una función (o método) pública(o) con el nombre 'connection()', de forma estática (Se usa 
                         directamente el método al llamar la Clase, no es necesario crear un objeto) y cierra el bloque en la línea 12. Esta función 
                         retorna un objeto de la Clase 'PDO' con la cadena de conexión a la base de datos.

                    D.   De las líneas 04 a la 08, se almacenan en variables los datos solicitados en una conexión a la Base de Datos 'MySQL'.

                    E.   En la línea 09, se crea una instancia de la Clase 'PDO' y como parámetros se le pasa al Constructor de la Clase la sintaxis 
                         de la cadena de conexión a la Base de Datos 'MySQL'.

                    F.   En la línea 10, se utiliza el método 'setAttribute()' que, recibe los posibles errores que surjan al intentar conectar 
                         con la Base de Datos 'MySQL'.

                    G.   En la línea 11, se retorna un objeto '$pdo' de la Clase 'PDO' con la cadena de conexión a la base de datos.
                |                                                                                                                                      |
                └ ------------------------------------------------------------------------------------------------------------------------------------ ┘

            03. Modificar al archivo 'index.php' con el siguiente código:
                    
                    01   <?php
                    02     require_once "models/DataBase.php";
                    03     if (!isset($_REQUEST['c'])) {
                    04       require_once "controllers/Landing.php";
                    05       $controller = new Landing;
                    06       $controller->main();
                    07     } else {        
                    08       $controller = $_REQUEST['c'];
                    09       require_once "controllers/" . $controller . ".php";
                    10       $controller = new $controller;        
                    11       $action = isset($_REQUEST['a']) ? $_REQUEST['a'] : 'main';
                    12       call_user_func(array($controller, $action));
                    13     }
                    14   ?>

                ┌ ------------------------------------------------------------------------------------------------------------------------------------ ┐
                |                                                                                                                                      |
                  EXPLICACIÓN:   
                    
                    ⦿   El código de este archivo, permite iniciar la aplicación y enrutar a un controlador, dependiendo de los datos que se envíen 
                         por la 'URL', donde 'c' es el controlador (Clase) y 'a' es la acción (función o método).

                |--------------------------------------------------------------------------------------------------------------------------------------|

                    $ git config --global user.name "nombre de su cuenta"

                    $ git remote add origin https://github.com/SenaProfeAlbeiro/ej01_n_capas_php_sql_html.git
                    $ git add .
                    $ git commit -m "Subiendo Proyecto"

            05. Click derecho sobre el área de trabajo y seleccionar la opción 'Open Git Bash here'; si no se encuentra la opción, hay que instalar

                    $ git pull

|-- AX_03.  Cambiar los puertos de 'Apache' en 'XAMPP': -------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|
                         
            01. En el Panel de control de XAMPP y en la misma línea del servicio 'Apache' dar click a 'Config / Apache (httpd.conf)'.
            
            02. Se abrirá un 'Bloc de Notas'. Dar click a la opción 'Edición / Buscar...' (o presionar las teclas 'CTRL + B').
            
            03. En la ventana emergente y en el control de texto 'Buscar: ', escribir '80' y dar click a 'Buscar siguiente'.
            
            04. Reemplazar todos los valores donde se encentre el puerto '80' con el puerto nuevo de trabajo, por ejemplo, '8080'. Dar click en 
                'cancelar' y guardar los cambios en el archivo.

            05. En el Panel de control de XAMPP dar click en 'start' de 'Apache' para iniciar el servicio.
            
            06. Comprobar que quedó de la forma correcta a través del navegador, con el siguiente enlace:

                         http://localhost:8080/ej01_n_capas_php_sql_html/

|-- AX_04.  Cambiar los puertos de 'MySQL' en 'XAMPP': --------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. En el Panel de control de XAMPP y en la misma línea del servicio 'MySQL' dar click a 'Config / my.ini'.

            02. Se abrirá un 'Bloc de Notas'. Dar click a la opción 'Edición / Buscar...' (o presionar las teclas 'CTRL + B').
            
            03. En la ventana emergente y en el control de texto 'Buscar: ', escribir '3306' y dar click a 'Buscar siguiente'.

            04. Reemplazar todos los valores donde se encentre el puerto '3306' con el puerto nuevo de trabajo, por ejemplo, '3308'. Dar click en 
                'cancelar' y guardar los cambios en el archivo.

            05. En el Panel de control de XAMPP y en la misma línea del servicio 'Apache' dar click a 'Config / Apache (php.ini)'.

            06. Se abrirá un 'Bloc de Notas'. Dar click a la opción 'Edición / Buscar...' (o presionar las teclas 'CTRL + B').

            07. En la ventana emergente y en el control de texto 'Buscar: ', escribir '3306' y dar click a 'Buscar siguiente'.

            08. Reemplazar todos los valores donde se encentre el puerto '3306' con el puerto nuevo de trabajo, por ejemplo, '3308'. Dar click en 
                'cancelar' y guardar los cambios en el archivo.
        
            09. En el Panel de control de XAMPP y en la misma línea del servicio 'Apache' dar click a 'Config / phpMyAdmin (config.inc.php)'.

            10. Buscar la línea '$cfg['Servers'][$i]['host'] = '127.0.0.1;' y agregar el puerto nuevo '3308' de la siguiente forma:
                         
                         $cfg['Servers'][$i]['host'] = '127.0.0.1:3308';

            11. En el Panel de control de XAMPP dar click en 'start' de 'MySQL' para iniciar el servicio.

            12. Comprobar que quedó de la forma correcta a través del navegador, con el siguiente enlace:

                         http://localhost:8080/phpmyadmin/
