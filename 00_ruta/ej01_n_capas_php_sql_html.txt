|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 05.01.  Crear la base de datos 'MySql': -------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Abrir el navegador y escribir http://localhost/phpmyadmin/.
            
            02. Dar click en 'SQL' y escribir el siguiente código:

                    01   CREATE DATABASE DB_PROJECTO DEFAULT CHARACTER SET utf8 ;
                    02   USE DB_PROJECTO ;
                    03 
                    04   CREATE TABLE ROLES (
                    05     rol_codigo INT NOT NULL AUTO_INCREMENT,
                    06     rol_nombre VARCHAR(45) NOT NULL,
                    07     PRIMARY KEY (rol_codigo)
                    08   ) ENGINE = InnoDB;
                    08 
                    09   CREATE TABLE USUARIOS (
                    10     rol_codigo INT NOT NULL,  
                    11     usuario_codigo INT NOT NULL AUTO_INCREMENT,
                    12     usuario_nombres VARCHAR(100) NOT NULL,
                    13     usuario_apellidos VARCHAR(100) NOT NULL,
                    14     usuario_identificador VARCHAR(20) NOT NULL,
                    15     usuario_email VARCHAR(50) NOT NULL,
                    16     usuario_pass VARCHAR(200) NOT NULL,
                    17     usuario_estado TINYINT NOT NULL,
                    18     PRIMARY KEY (usuario_codigo),
                    19     INDEX ind_usuario_rol (rol_codigo ASC),
                    20     CONSTRAINT fk_usuario_rol
                    21       FOREIGN KEY (rol_codigo)
                    22       REFERENCES ROLES (rol_codigo)
                    23       ON DELETE CASCADE
                    24       ON UPDATE CASCADE
                    25   ) ENGINE = InnoDB;

            02. Dar click en 'Continuar'.
